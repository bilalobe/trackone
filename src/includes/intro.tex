\section{Introduction}
Heritage-safe monitoring requires reversible, low-impact sensing with ultra--low-power links. We present a deterministic telemetry pipeline for ``Barnacle'' pods that (i) secures data end-to-end, and (ii) commits daily proofs via OpenTimestamps without publishing payloads.

\textbf{Contributions:}
\begin{itemize}[nosep]
    \item A compact frame and key lifecycle (ADR-001, ADR-002)
    \item Deterministic canonicalization and Merkle policy with daily OTS anchoring (ADR-003)
    \item A reference gateway pipeline with tests and reproducible artifacts
\end{itemize}
